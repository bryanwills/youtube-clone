# YouTube Clone

**NPM Commands**

```shell
$ npm run <options>

OPTIONS:

start        - Starts the development server.
build        - Builds the app for production.
format:check - Checks the style of JS, JSX, HTML and SCSS code.
format:fix   - Fix the style of JS, JSX, HTML and SCSS code.
```
